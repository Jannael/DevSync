<Aside type="note">

El comando `build` está diseñado para ejecutarse en una GitHub Action, pero puedes ejecutarlo localmente si tienes todas las dependencias instaladas.

</Aside>

## Comandos Disponibles

<CardGrid>
<Card title="init" icon="rocket">

Copia la plantilla de portafolio predeterminada a tu carpeta actual.

```sh
bunx @jannael/devsync init
```

</Card>

<Card title="build" icon="setting">

Construye `linkedin.md`, tu `Perfil de GitHub`, `académicos`, y genera un PDF desde tu componente `CV`.

```sh
bunx @jannael/devsync build
```

</Card>

<Card title="create-template" icon="document">

Crea una nueva plantilla personalizada para Devsync.

```sh
bunx @jannael/devsync create-template
```

</Card>
</CardGrid>
