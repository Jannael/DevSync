## Build Command
The `build` command performs the following steps:

<Aside type="note">

This command is meant to run on GitHub Actions workflows, but it can also be run locally if you have Bun and Puppeteer installed.

</Aside>

<Steps>

1. Installs dependencies using `bun install`.

2. Builds the project using `bun run build`.

3. Validates the `DEVSYNC.json` file.

4. Builds the CV for each language.

5. Creates the LinkedIn profile for each language.

6. Creates the GitHub profile using the `default lang`.

7. Creates the academics profile using the `default lang`.

</Steps>
