This guide will help you get started with Devsync. Follow the steps below to create your portfolio, resume, GitHub profile, LinkedIn, and academics all at once.

<Steps>

1. Initialize a project using Devsync:

   ```sh
   bunx @jannael/devsync init
   ```

2. Create a repository with your username (example using GitHub CLI)

   ```sh
   gh repo create [username]/[username] --public
   ```

3. Add a remote origin:

   ```sh
   git remote add origin https://github.com/[username]/[username].git
   ```

4. Push your project

   ```sh
   git push -u origin main
   ```

5. Fill DEVSYNC.json with your information

6. Push your changes

</Steps>

Now you can update `DEVSYNC.json` to refresh your portfolio, resume, GitHub profile, LinkedIn, and academics all at once, whenever you want.

## Tips

- `DEVSYNC.json` is not set in stone; you can delete most of its fields if you don't need them, and they simply won't be displayed. However, keep in mind that the ability to generate a `portfolio` and `resume` while omitting certain fields depends entirely on the template you are using.
