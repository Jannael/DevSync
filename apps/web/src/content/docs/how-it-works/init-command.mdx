## Init Command
The `init` command performs the following steps:

<Steps>

1. First, it checks if the `--template` flag is provided. If it is, the command splits the string by `/` and cleans it up.

2. Clones the repository. If the `--template` flag is not provided, it clones the default template: `https://github.com/jannael/devsync-default-template`.

3. Deletes the `.git` directory to ensure a clean Git history, separate from the template's history.

4. Initializes a new Git repository and makes the first commit, allowing the user to add their own remote origin.

</Steps>
