<Aside type="tip">
I personally recommend using a serverless hosting like vercel, cloudflare pages, or netlify to host your portfolio. These platforms are free and easy to use, and they automatically deploy your changes when you push them to your repository. You can also use a custom domain to make your portfolio more professional.
</Aside>

This guide will help you get started with Devsync. Follow the steps below to create your portfolio, resume, github profile, linkedin and academics all at once.

<Steps>

1. Init a project using devsync:

   ```sh
   bunx @jannael/devsync init
   ```

2. Create a repository with your username (example using GitHub CLI)

   ```sh
   gh repo create [username]/[username] --public
   ```

3. Add remote origin

   ```sh
   git remote add origin https://github.com/[username]/[username].git
   ```

4. Push your project

   ```sh
   git push -u origin main
   ```

5. Fill DEVSYNC.json with your information

6. Push your changes

</Steps>

Now you can change DEVSYNC.json to update your portfolio, resume, github profile, linkedin and academics any time you want all at once.
